\# 🕵️ Fake Review Detector



\## 📌 Project Description

This project detects whether a customer review is Genuine or Fake using Machine Learning.



\## 🚀 Technologies Used

\- Python

\- Streamlit

\- Scikit-learn

\- Pandas

\- Joblib

\- NumPy



\## 📂 Files

\- app.py - Streamlit application

\- model.py - Model training

\- reviews.csv - Dataset

\- model.pkl - Trained model

\- vectorizer.pkl - Text vectorizer

\- requirements.txt - Required packages



\## ▶️ How to Run



```bash

streamlit run app.py

```



\## 📷 Output



Enter a review and click \*\*Check\*\* to predict whether it is Fake or Genuine.
